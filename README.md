## ✨ ویژگی ها

- 🤖 **هوش مصنوعی قدرتمند** - اتصال به Groq API با مدل LLaMA 3.3 70B
- 🎨 **رابط کاربری شبیه ChatGPT** - با طراحی مذهبی-مسجدی
- 💾 **ذخیره تاریخچه گفتگوها** - در مرورگر کاربر (LocalStorage)
- ⚙️ **پنل مدیریت کامل** - با دسترسی از طریق رمز عبور
- 📱 **ریسپانسیو** - سازگار با موبایل و دسکتاپ
- 🔐 **امنیت کامل** - API Key از طریق Cloudflare Worker مخفی می‌شود
